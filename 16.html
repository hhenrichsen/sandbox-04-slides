Accounts and Organizations

Let's Build a Code Snippet Platform

Requirements

  • Allow users to sign up and sign in
  • Users should be able to create snippets, which contain 1 or more code block "files" inside
  • Users should be able to add other users that can edit snippets
  • Users should be able to comment on snippets
  • Users should be able to list snippets created by a specific user
  • Users should be able to mark snippets as private

Bonus:

  • Users should be able to create organizations which contain 1 or more users
  • Users should be able to create snippets within an organization
  • Users should be able to list snippets created by an organization
